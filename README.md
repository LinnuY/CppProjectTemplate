## Install

This project uses xmake build.

```shell
git clone https://github.com/LinnuY/cppDemo.git ./cppDemo
cd cppDemo
xmake build
xmake install
```

If you don't install xmake, please install it.

### Arch or Manjaro

```shell
yay -S xmake
```
